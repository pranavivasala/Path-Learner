# AI Learning Path Generator

This project is a simple and practical tool designed to make learning AI more structured and less overwhelming. Instead of searching randomly for resources, users get a clear, personalized path they can follow step by step.

## What it does

The application helps users generate a customized AI learning journey based on their goals. It breaks learning into daily tasks, suggests useful resources, and allows users to track their progress over time.

## Key Features

* User authentication with secure login and signup
* Personalized learning paths based on user goals
* Day-by-day learning structure for better consistency
* Curated resources for each topic
* Progress tracking and data saving
* Resume learning anytime without losing progress

## Tech Stack

* **Core Framework:** Python, Streamlit
* **Database:** SQLite
* **AI Model:** Google Gemini API

### Libraries Used

* `bcrypt` – for secure password hashing
* `pandas` – for managing and displaying data (admin dashboard)
* `plotly` – for progress visualization and charts

## Why this project

Learning AI can feel confusing because of the number of resources available. This project focuses on clarity and consistency by providing a structured path and keeping everything in one place.

## Future Improvements

* Smarter AI-based recommendations
* Detailed progress analytics
* Gamification features like streaks or badges

This project is built to make learning more organized and manageable, especially for beginners trying to get into AI.
